## About This Project

This is my Digital Literacy Portfolio, created as part of the CSE0001 course at VIT Bhopal University. The goal of this project is to help us as first year students understand how to use digital tools responsibly, build a professional online presence, and stay safe on the internet. I have organised all my work into separate folders, one for each task.

---

## Module Summaries

### Task 1 – Digital Literacy Infographic
For this task I made a one page infographic explaining what digital literacy is, some useful tools for students, and how to build a professional online presence. I used Canva to design it since it has a lot of free templates and is easy to use.

### Task 2 – Student Digital Portfolio
In this task I created and set up profiles on LinkedIn, GitHub and Kaggle. I had not used LinkedIn properly before this so it was a good experience to fill in my education details and understand what the platform is actually used for.

### Task 3 – Coding and Collaboration Platforms
I created an account on HackerRank and completed a beginner level challenge. I also built a 5 question Digital Literacy Quiz using Google Forms which can actually be used to test my batchmates.

### Task 4 – Email Etiquette
I wrote two professional emails, one to a professor requesting a deadline extension and one to an internship coordinator. I also made a social media do's and don'ts checklist for college students.

### Task 5 – Cybercrime Awareness
For this task I researched phishing attacks and wrote a case study explaining how they work and who they target. I also created a prevention checklist with tips specific to college students in India, including UPI safety tips.
